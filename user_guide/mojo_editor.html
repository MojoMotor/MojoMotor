MojoMotor Editor

The MojoMotor Editor is the heart and soul of content editing within the system. Any Region can be simply edited. When the Mojo Toolbar is exposed, regions are clearly identified for you:

To put a region into edit-mode, simply click into it.

Enabling HTML source editing

There are 2 ways the editor can be set up to edit the raw HTML instead of using the default WYSIWYG toolset.

  1. To permanently edit in this mode, use the "Account" link, and set your "Edit Mode" to "Plain Text". If at any time you wish to return to WYSIWYG, simply toggle back to "Graphical". Changes to your edit mode will be reflected the next time you go to edit a region.
  2. To enable temporary source code editing, you can add the 'Source' option into the editor toolbar. See system/mojomotor/config/ckeditor.php for examples. This change affects all members.
