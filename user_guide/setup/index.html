Setup Overview

The setup phase (installer, importer and updater) are concerned with how the end-user will get MojoMotor set up on their servers, and preparing them to use the software. Installer will only need to be run for new installations, and will automatically transition into the update/importer phase if appropriate. The importer’s role is to bring content generated outside the system into MojoMotor’s control, for use within the software. The principal role of the updater is for feature enhancements and security releases. I’d like to test the feasibility of a one-click auto-updater (no download/ftp).

Setting up MojoMotor is very easy. The 3 tasks that you can do from setup are:

  • Requirements
  • Installing MojoMotor (also, see alternate installation instructions)
  • Importing an existing website (and a related case study)
  • Updating to a newer version of MojoMotor
