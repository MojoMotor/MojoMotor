Installing MojoMotor

Requirements Checklist

We have created a Server Wizard that will verify whether your server is compatible with MojoMotor. To use the Wizard:

  • Download and unzip the archive.
  • Upload the folder to your server.
  • Point your web browser to the folder.  Example: http://example.com/mm_server_wizard/
  • PHP 5.1.6 or higher is required by MojoMotor
  • MojoMotor requires a MySQL database
    • MojoMotor will attempt to create a database for you if it cannot find the one you specify, however some servers disallow this, and you will need to create it before you install.
  • MojoMotor will need to make some files such as config, database and routes in system/mojomotor/config writable. If your server doesn't allow PHP to change files to "writable" (also known as "chmod") you'll need to follow the alternate Installation instructions.

Supported Browsers

The sites you build in MojoMotor are purely your own HTML, CSS, Javascript, and other assets. They will work in any browser you target. In other words, what your site visitors see, and what your end site will support is totally in your hands. The MojoMotor features however, such as the Admin Toolbar or the MojoMotor Editor use technology that requires modern browsers. If you or your client needs to use MojoMotor, a modern browser with Javascript and CSS enabled is required with at least a 1024 pixel viewport. The following browsers are known to work:

  • Firefox 3+
  • Safari 4+
  • Chrome
  • Internet Explorer 8+
