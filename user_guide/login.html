MojoMotor Login

There are 2 ways to log into MojoMotor. If "In page login" is enabled in Site Settings, then it is possible to use {mojo:site:login} in any layout, and have a modal login box on any page.

If in page login is not enabled, then you can access the login page by visiting http://example.com/index.php/login/

Forgotten Passwords

If you have forgotten your password, MojoMotor can reset it for you via the "forgotten password" feature. From any login (modal or full page) you are presented with a "forgotten password?" link. If you follow it, MojoMotor will ask for your email. Enter it, and you will be guided through resetting your password from there.
